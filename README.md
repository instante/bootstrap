Requirements
------------

- PHP 5.5 or higher

- [Nette Framework](https://github.com/nette/nette) 2.2



Installation
------------

The best way to install Instante bootstrap is using  [Composer](http://getcomposer.org/):

```
$ composer require instante/bootstrap
```
